🚀 Introdução às Funções em Python

Este repositório contém exemplos práticos sobre o uso de funções em Python, abordando desde conceitos básicos até o uso de parâmetros e valores padrão.

📚 Sobre

Funções são blocos de código reutilizáveis que permitem executar uma ação específica sempre que necessário. Elas ajudam a evitar repetição, tornam o código mais organizado e facilitam a manutenção.

Por padrão, funções em Python retornam None quando não possuem um return.

🧠 Conceitos abordados
Criação de funções com def
Execução de funções múltiplas vezes
Parâmetros (argumentos)
Erros ao não passar argumentos
Valores padrão em funções
